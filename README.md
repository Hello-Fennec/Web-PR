# HelloWorld 2022 Web-PR 


### React Project built on Vite JS


run command: `pnpm run dev`

build command: `pnpm run build`

### Installed packages

* `Tailwind css` 
* `HowlJS` for playing audio
* `React Icons`

### Custom hooks

* `useWindowDimensions` to get window width and height
